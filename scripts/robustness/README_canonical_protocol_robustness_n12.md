# Canonical CIFAR-100 protocol-robustness n=12 scripts

These scripts are patched from the n=6 ADS/TFS scripts for the separate
canonical-checkpoint robustness analysis.

Canonical conditions:
- `alibi` = 1D-ALiBi canonical
- `alibi_2d` = 2D-ALiBi canonical
- `alibi_2d_matched` = 2D-ALiBi matched

Canonical seeds:
`1 5 7 11 13 21 42 99 123 456 2024 31337`

Important implementation convention:
- Checkpoint/result labels remain `alibi`, `alibi_2d`, `alibi_2d_matched`.
- `alibi_2d_matched` is instantiated with model architecture `pe_type='alibi_2d'` via `MODEL_PE_ALIAS`.
- Checkpoint folders are still expected as `<condition>_seed<seed>/best_model.pth`.

Primary protocol-robustness scripts for the CIFAR-100 canonical set:
- `ads_experiment_cifar_protocol_robustness_n12.py`
- `ads_specificity_cifar_protocol_robustness_n12.py`
- `ads_probing_residual_cifar_protocol_robustness_n12.py`

The other patched files are included for consistency/permissive condition-label support, but adaptive/ref-evasion analyses need calibrated benign/noise-floor thresholds before being used as publication-grade canonical robustness results.

Suggested output root:
`/content/drive/MyDrive/ads_protocol_robustness_cifar_n12/data`

Smoke test:
```bash
python -u /content/ads_experiment_cifar_protocol_robustness_n12.py \
  --models_dir "/content/drive/MyDrive/pe_experiment/results_cifar100_v2" \

```

Full main ADS run:
```bash

  --models_dir "/content/drive/MyDrive/pe_experiment/results_cifar100_v2" \
  --val_dir "/tmp/cifar100" \
  --output_path "/content/drive/MyDrive/ads_protocol_robustness_cifar_n12/data/ads_results_cifar100_canonical_n12.json"
```



# v2 model definition

This package includes `full_scale_experiment_v2.py`, generated by applying the 2D-ALiBi patch to the uploaded base `full_scale_experiment.py`.

The ADS protocol-robustness scripts import:

```python
from full_scale_experiment_v2 import VisionTransformer
```

This is required because the unpatched `full_scale_experiment.py` accepts only the original PE types and raises:

```text
ValueError: Unknown PE type: alibi_2d
```

For the CIFAR-100 canonical robustness analysis, run only:

```text
ads_experiment_cifar_protocol_robustness_n12.py
ads_specificity_cifar_protocol_robustness_n12.py
ads_probing_residual_cifar_protocol_robustness_n12.py
```

Copy these four files to `/content` in Colab:

```text
full_scale_experiment_v2.py
ads_experiment_cifar_protocol_robustness_n12.py
ads_specificity_cifar_protocol_robustness_n12.py
ads_probing_residual_cifar_protocol_robustness_n12.py
```

## Figure typography update

The CIFAR ADS plotting routine now uses 13 pt typography throughout generated figures.
For the dual-axis early-warning panels, the ADS/Accuracy legend is placed at the
upper center above each subplot (`loc=lower center`, `bbox_to_anchor=(0.5, 1.02)`,
`ncol=2`) so it no longer covers the curves. Figure export DPI is set to 300.
